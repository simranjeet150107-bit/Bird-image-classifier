# 🐦 Bird Species Classifier using Transfer Learning

## 📌 Project Description

This project is a Bird Species Classification system developed using Python and Deep Learning. It uses Transfer Learning with a pretrained EfficientNet model to identify bird species from images.

The model is trained on the Kaggle **100 Bird Species Dataset** and can classify uploaded bird images into their respective species.

This project demonstrates image preprocessing, transfer learning, model training, evaluation, and prediction using PyTorch.

---

## ✨ Features

- Bird image classification
- Transfer Learning using EfficientNet
- PyTorch implementation
- Image preprocessing
- Model evaluation
- Predict bird species from uploaded images

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/bird-species-classifier.git
```

Move into the project folder

```bash
cd bird-species-classifier
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Open the Jupyter Notebook

```bash
jupyter notebook
```

Run:

```
bird_species_classifier.ipynb
```

## 📊 Model

- Architecture: EfficientNet
- Framework: PyTorch
- Image Size: 224 × 224
- Loss Function: CrossEntropyLoss
- Optimizer: Adam
